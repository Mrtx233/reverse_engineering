
对称加密

对称加密的逻辑:
    加密和解密的秘钥是相同的.
    最常见的对称加密:
        AES, DES, 3DES加密
    加密算法我们是不用研究的.
    各个语言都有加密算法的封装. 都是标准算法, 标准库

python语言的对称加密
    需要安装pycryptodome库
    pip install pycryptodome

    有极个别同学的电脑可能安装好了. 用不了.
    请检查. 安装路径的大小写.
    装好了之后, 该库的名称是:  Crypto, 对应的文件夹也是Crypto
    如果你的文件夹是crypto, 请手动把它改成Crypto

    加密过程:
        1. 创建加密器
            aes = AES.new(key=key, iv=iv, mode=AES.MODE_CBC)
        2. 对数据进行填充
            ming_bs = pad(s.encode("utf-8"), 16)
        3. 对数据加密
            mi_bs = aes.encrypt(ming_bs)
        4. 对密文进行base64编码
            b64_str = base64.b64encode(mi_bs).decode()

    解密:
        1. 创建加密器
            aes = AES.new(key=key, iv=iv, mode=AES.MODE_CBC)
        2. base64对数据进行解码
            mi_bs = base64.b64decode(s)
        3. 解密
            ming_bs = aes.decrypt(mi_bs)
        4. 去填充
            ming_bs = unpad(ming_bs, 16)
