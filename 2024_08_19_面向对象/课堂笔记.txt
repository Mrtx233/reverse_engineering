面向过程的开发思路:
    一切以流程为核心.

面向对象的开发思路:
    一切以对象为核心.
    何为对象?
        对象就是能帮你干活的东西...
        示例1,
            要把大象装冰箱. 总共分几步?
            1. 把冰箱门打开
            2. 把大象装进去
            3. 把冰箱门关上

            面向对象的思路:
                大象. 你进冰箱
                大象自己去开门
                大象自己装自己
                大象自己把门关上
            在这个逻辑里面. 你只需要找到一个可以帮你完成把自己冰箱里的大象就可以了
        示例2,
            我想喝可乐.
            1. 面向过程:
                1. 从沙发上起来. 走到冰箱门口
                2. 把可乐拿出来
                3. 把冰箱门关上.
                4. 回到沙发喝可乐

            2. 面向对象:
                你找个对象.
                    命令你的对象. 去拿可乐.
                    你的对象. 去冰箱
                    你的对象, 拿可乐
                    你的对象, 交到你手里
                    你, 喝可乐
                在这个模型当中, 你只需要找到一个可以帮你拿可乐的对象就可以了.
        在上述两个示例中, 你只需要找到一个可以帮你完成某个工作的对象就可以了

        如何创建对象呢?
            你需要先写类.
            用类来描述, 未来你创建出来的对象, 是什么样子的.

            然后, 你可以用类. 来创建对象

            类: 用来描述对象的.
            对象: 用类来创建出来. 给你干活的东西

            在面向对象的世界里. 对象才是干活的那一个.
            出了问题. 先思考对象, 然后再思考类....

        类的语法:
            类也是对一类事物的总结和归纳
            class 类名:
                类里面的东西
                def xxxx():  # 类中的函数, 被称为方法, 是给对象使用的
                    pass
        创建对象:
            obj = 类名()
            obj.类中定义的方法()

        方法参数的self是什么?
            当前正在执行该方法的对象.
            张三.砍人()  # self是张三
            李四.砍人()  # self是李四

        属性:
            给对象设置属性. 属性名字随便定义, 属性值随意...
            对象.属性 = 值

            使用属性:
            对象.属性
        __init__
            对象的初始化方法, 在对象被创建的时候. 自动的
            执行该方法. 我们可以把初始化对象的逻辑写在该方法内
            通常:
                def __init__(self, name, age, shengao):  # 给新创建的对象初始化
                    # 当前对象的名字 = 你传过来的名字
                    self.name = name
                    self.age = age
                    self.shengao = shengao
        面向对象的三大特征:
            1. 封装
                把一类事物封装在一个类中.
            2. 继承
                子类可以自动拥有父类的除了私有内容以外的所有内容
            3. 多态
                一个对象, 多种形态.
                python的多态体现的是鸭子模型
                鸭子模型, 蒙上眼睛, 只要会嘎嘎叫的, 就可以认为是鸭子
